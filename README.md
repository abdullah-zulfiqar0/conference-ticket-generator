Frontend Mentor - Conference Ticket Generator Solution

This is a solution to the Conference Ticket Generator challenge on Frontend Mentor. This project helped me practice form validation, DOM manipulation, file upload handling, and responsive layout using basic frontend technologies.

Overview
The challenge

Users should be able to:

Complete a form with their Full Name, Email Address, and GitHub Username

Upload an avatar (JPG or PNG, max 500KB)

Receive validation error messages for:

Empty fields

Invalid email format

Invalid file type or file size

Generate a personalized conference ticket after form submission

View a responsive layout on both mobile and desktop screen sizes

See hover and focus effects on interactive elements
