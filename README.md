<h1>Frontend Mentor - Conference Ticket Generator Solution</h1>

<p>This is a solution to the Conference ticket generator challenge on Frontend Mentor. This project helped me practice form validation, DOM manipulation, file upload handling, and responsive layout using basic frontend technologies.</p>

<h2>Overview</h2>

<h3>The challenge</h3>
<p>Users should be able to:</p>
<ul>
  <li>Complete a form with their Full Name, Email Address, and GitHub Username</li>
  <li>Upload an avatar (JPG or PNG, max 500KB)</li>
  <li>Receive validation error messages for:
    <ul>
      <li>Empty fields</li>
      <li>Invalid email format</li>
      <li>Invalid file type or file size</li>
    </ul>
  </li>
  <li>Generate a personalized conference ticket after form submission</li>
  <li>View a responsive layout on both mobile and desktop screen sizes</li>
  <li>See hover and focus effects on interactive elements</li>
</ul>
